# 14. チャンピオン収益+10%への挑戦 — 当たりはサイジング指数1つ、偽レバー3.5本の解剖

日付: 2026-06-11 / 対象: `confluence_meanrev_v2` + 最適資金管理([[09]]) / 制約: **金を含む銘柄は使わない**

**問い(ユーザー)**: チャンピオン手法の収益を10%以上改善せよ(金銘柄の取引以外の方法で)。

**結論(先出し)**: 達成。採用した変更は **z-power サイジングの指数 P: 2.0 → 4.0 の1パラメータのみ**。
同一テール基準(ブートストラップ p95 DD=20%較正)で CAGR **+15.04% → +17.40%(+15.7%相対)**、
empirical 20% 較正でも **+23.83% → +27.63%(+15.9%相対)**、理論テール p95 はフラット(-29.4→-29.2%)
= レバ偽装ではない。敵対検証7項目(独立再実装/IS単独選択テスト/5シード/配分集中/ブロック長/mp依存/年次)
を全通過。併せて、候補4レバーの徹底検証で **「キャリー保険はリークだった」という reports/10 の訂正事項**を
含む重要な負の知見を3.5本確定した。

  robust seed0 **+15.21%**(k=7.57)。

  (empirical↑かつ p95 悪化は不採用)+ IS較正→OOS維持 + 高原。

## 1. 採用 — z-power 指数 P の再較正(P=2.0→4.0)

## 2. 棄却された偽レバー(負の知見3.5本)

### 2.1 キャリー・スリーブ — **致命的リークの発見**(exp34、reports/10 の訂正事項)
ens_lab 統合+robust 再較正でキャリー(h42 legs2 w0.25)は表面上 **mean3 +16.81%** で合格ラインを超えた
(p95 5pp改善・高原滑らか・全年プラス)。しかし必須のリーク監査 —— **金利を1年ラグして順位付け** ——
で崩壊: robust +14.16%(ベースライン未満)、emp p95 悪化(-32.6%)、失血窓ヘッジの2022年寄与 +0.28→-0.03 に消滅。
**`fxlab/carry.py` の暦年平均政策金利は「年初時点では未確定の利上げ情報」を含む**(特に2022利上げサイクル)。
→ **[[10]] の「キャリー=本物の保険(in_bleed +420/月, IS/OOS持続)」は同リークの上の結論であり要訂正**。
実勢スワップの月次実データが入手できた場合のみ再挑戦の価値あり。

### 2.2 通貨ファクター・ネッティング上限 — 構造的に不成立(exp35)
新規建玉時に通貨別正味エクスポージャへ cap を課す完全未踏のレバー。8構成(USD/全通貨 × γ1〜3)が
**全て robust +11.4〜14.0% でベースライン未満**、γ→∞ へ単調回復=「最良の上限は上限なし」。
capは2022型共通モードだけでなく、z-power が意図的に厚くした深乖離トレード(利益源)を優先的に切る。
**通貨集中はテールの「署名」であって、入口の建玉縮小で外科的に除去できる「原因」ではない**(exp30と同型)。

### 2.3 DD連動トレンド・クッション×mp連立 — robust +17.4% は出るが「2022 n=1」(exp36/39)
[[10]]§5 の唯一の収穫(クッション)に「mp連立」の未検証相互作用が実在: mp単独は前進ゼロだが、
クッション併用で robust が mp と単調増加し mp15 で **mean5 +17.12%**、合成(P3.5×クッション)では
**+19.79%** まで出る。**しかし敵対検証で核心が2つ崩れた**:
(a) overlay 9近傍のうち成立は全期間データで選抜済みの(30/100)列のみ、(20/50)/(40/150)の6点は
ベースライン以下(最悪は有害)。donchian 交換も崩壊。
(b) **2022除外でベースラインに逆転負け**(+13.33% vs +14.00%)。IS期(2017-21)の発火25件は累計**-3.19%**。
= robust優位は全額「2022の一撃が較正窓に入っていること」に担保されており、**forward期待CAGRに計上不可**。
「平時は負キャリーの2022型危機保険(期待リターン寄与ゼロ)」としての任意装備なら経済的に整合的
(テール圧縮 p95 -29.4→-24% は2022除外でも構造的に残る)。採否はリスク選好の問題であり収益改善ではない。

### 2.4 ER連動グレーディング — 勾配が存在しない(exp37)
ER五分位×プールret/PF の勾配は仮説と逆符号かつ微弱(spearman +0.067)、IS/OOSで形が反転。
低ER増し6構成は全て「empirical↑+p95悪化」のレバ偽装署名、診断方向(高ER増し)は単純にCAGR純減。
**ERは入口の二値フィルタ([[08]])としてのみ有効で、サイズの連続情報は持たない**。

## 3. プロトコルへの追加(恒久)

- **IS単独選択テスト(IS-argmax監査)**: パラメータ再調整の改善主張には「ISデータだけを見た研究者は
  どの値を選んだか+その値のOOS成績」を必須化。逐次探索バイアスを直接暴く(§1で有効性実証)。
- **暦年粒度のファンダメンタル系列はラグ監査必須**: 年次近似値は「その年の将来」を含む(§2.1の判例)。
- ペアシード比較(`tail_protocol.py`)を robust 較正の標準とする。
