# レポート08: ワーストトレード解剖 → エントリーフィルタによる底上げ(v2)

日付: 2026-06-07 / 前報: [07](07_production_spec.md)
動機: ユーザー要望「最も成績の悪いトレードの正体・共通点・他との相違を細かく分析し、それに引っ掛からない改善を実施」。

## 1. ワーストの正体 = 長期保有の「塩漬け」(出口では直せない)

19対象横断・全1232トレードを解剖(exp12):
- **`corr(保有バー数, リターン) = -0.85`**。保有0-15本は勝率99-100%でプラス、31-60本は勝率11%で-$16.3k、61本以上は勝率0%。
- **ワースト10%トレード(平均保有55本=全体の2.5倍)が総損失の72.5%**。2022年(利上げトレンド)に集中、ショート寄り。

出口改善は全滅(exp13/14, 14通り×IS/OOS):損切り(sl)・時間ストップ(max_bars)はいずれも **リターン・Sharpe・最大DD すべて悪化**。平均回帰では損切りが「戻る直前の含み損」を実現損に変え、ストレス期に実現損がクラスタしてDDを増やす。**塩漬けの失血=反転を待つ忍耐=エッジそのものの対価**。→ 残る打ち手はエントリー側のみ。

## 2. エントリー側を多角検証(Workflow: champion-entry-filter, 15エージェント)

エントリー時点の因果特徴量(先読みなし)を全トレードに付与(`filterlab.py dataset` → `results/entry_features.csv`)し、5レンズで「カタストロフは入る前に予測可能か」を分析 → 仮説 → IS/OOS+高原+敵対検証。

**分離可能性の正直な結論:** 個別トレード単位ではカタストロフは**ほぼ予測不能**(最良特徴量 er40 でも AUC 0.558、並べ替え検定 p=0.27=有意でない)。しかし**ポートフォリオ/エクイティ曲線レベルでは、効率比フィルタが純損失レジームを一括除去して効く**。

**汎用・非カーブフィットで効くフィルタは1つだけ:**

> ★ **Kaufman 効率比 ER(40) ≤ 0.55 のときだけ建玉**(`er_win=40, er_max=0.55`)
> ER = |Δ40本| / Σ|Δ1本| ∈ [0,1]。高=一直線の強トレンド(逆張りの落ちるナイフ)、低=往復レンジ(平均回帰の好地合い)。

棄却された罠(すべて実測で確認): 損切り/時間ストップ(出口)、`slow_z_cap`(深い乖離=エッジ源を削る/-22% pnl)、`atr_cap`、`slope_max`、`adx`(er40の上に足すと劣化=死荷重)。

## 4. 正直な限界

- これは**コホート単位のリスク低減であって、テール上限ではない**。単発ワースト(NZDCAD ショート -7.55%, 140本)はER中位で入っており、どのしきい値でも除外されない。**塩漬けの"発生そのもの"をエントリーで根絶するのはカーブフィット無しには不可能**=平均回帰エッジの不可避コスト([05]鉄の三角形と整合)。
- PF集計2.0は依然未達(1.91)=鉄の三角形は不変。改善は「同条件下での底上げ」。

## 5. 結論 / 成果物

**新チャンピオン `strategies/confluence_meanrev_v2.py`**(v1 + ER40≤0.55、他フィルタOFF固定)を確定。リターン・PF・Sharpe・最大DD を**フルもOOSも**改善し、毎年プラス・年100取引・非カーブフィット・先読みなし・全26テスト緑。再現: `uv run python portfolio.py confluence_meanrev_v2 --tf H4 --exclude AUDJPY`。検証コード: exp12/13/14・filterlab.py・strategies/confluence_meanrev_filtered.py。
