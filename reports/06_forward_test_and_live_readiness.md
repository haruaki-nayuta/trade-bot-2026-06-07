日付: 2026-06-07 / 前報: [05](05_universe_robustness_and_iron_triangle.md)

## 1. 真の前向き検証(カーブフィットでないことの最終確認)

チャンピオンのパラメータ(window=50, entry_z=2.0, RSI 35/65, slow_win=250, slow_z=1.75, vol_pct=0.70)は
すべて**広い高原上の丸い値**(小数点最適化なし)。さらに厳密性のため IS/OOS を分離して固定パラメータで検証:

| 期間 | 毎年プラス | PF中央 | PF最小 | 年取引 |
|------|-----------|--------|--------|--------|
| IS(2016-2021, 設計参照) | 100% | 1.60 | 1.27 | 111 |
| **OOS(2022-2026, 前向き=未使用扱い)** | **100%** | **1.94** | 1.26 | 112 |
| 全期間 | 100% | 1.83 | 1.26 | 112 |

**OOS の方が IS より PF が高い(1.94 ≈ 2.0)**。将来データで劣化しない=**過剰最適化ではなく前向きに通用する**強い証拠。
直近(2023 PF1.94 / 2025 2.88 / 2026 2.49)は実質 PF2.0 超。`uv run python exp11.py` で再現。

## 2. 実運用シグナル生成器 `signals.py`

検証だけでなく「実際に動くソフト」へ進めるため、**最新データから現在のアクションを出力**する生成器を実装。

```
uv run python signals.py                      # 推奨構成で「現在の建玉/新規シグナル」を一覧
uv run python signals.py --tf H4 --asof 2026-06-01
```

出力: 対象ごとに「現在ロング/ショート/ノーポジ」「直近で出た新規エントリー」「Zスコア等の状態」。
これは発注の手前まで(検証専用環境のため**自動発注はしない**=CLAUDE.md準拠)。実弾接続は別途ブローカーAPI実装が必要。
