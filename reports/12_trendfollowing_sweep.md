# 12. トレンドフォロー網羅検証 — 7族×3時間足×67構成、生存ゼロ

日付: 2026-06-11 / 前報: [11](11_2x_attempt_structures.md) / 動機: ユーザー要望「トレンドフォローの手法も試してみて」

**結論(先出し)**: トレンドフォロー7族(ドンチャン/MAクロス/時系列モメンタム/ADX・ERゲート型/
ボラブレイク/押し目トレンド/クロスセクショナル・モメンタム)× H4/D1/W1 の**固定グリッド67構成を
並列バックテスト(Workflow 7エージェント)した結果、生存基準(IS と OOS の両方で PF>1.0)を
満たす構成はゼロ**だった。大半は IS の時点で負けており(カーブフィット崩壊ではなくエッジ不在)、
1トレードあたりの負け幅(−5〜−73bps)はコスト(1〜4bps)を大きく超える=**シグナル自体が逆向き**。
FX メジャー/クロスの H4〜W1 横断では「ブレイク/トレンド確認後の順張りは高値掴み・安値売りになる」
という平均回帰優位の構造が確認された。レポート01/02(ベースライン地形)・10(補完オーバーレイ棄却)と
完全に整合し、トレンドフォローの単体採用・統合追加とも**棄却**。

- ユニバース: 19対象(メジャー7+クロス12, AUDJPY除外)、コスト計上(クロス3pips)、size=value。

- 生存基準: IS(〜2021)と OOS(2022〜)の両方で PF>1.0 かつ n≥100 → 生存者は

## 2. なぜ負けるのか(価格構造の解釈)

- 負けの主因はコストではない: mean_bps −5〜−73 に対しコスト寄与は 1〜4bps。**トレンド確認後の
  エントリーが平均回帰の反転点を掴む**(=チャンピオンが拾っている反転の「反対側」)。
- 時間足を上げる(D1→W1)ほど1トレードの負けが深くなる=長い horizon でも FX メジャーの
  持続トレンドは「確認してから乗る」には遅すぎる。
- 2022以降だけ僅かに改善する族(クロスセクショナル、押し目D1)があるが、金利差レジームの
  追い風込みでも PF1.0 前後止まり。

## 3. 統合(混ぜる)可能性について

単体黒字のトレンドストリームが存在しないため、統合の前提を満たさない。なお:
- 単体赤字のトレンドを「失血窓の保険」として混ぜる案は [10] で11エージェント検証済み=
  **統合DD=20%で純増ゼロ**(保険料がDD便益を上回る)。
- 本セッションの同一テール・プロトコル([11])でも、empirical だけ改善する構成はレバ偽装として棄却される。
- 唯一の残存オプションは [10]§5 の **DD連動ADXショート・オーバーレイ=CAGR中立のテールクッション**
  (p95 −28.7→−23.7%)。利益目的では無価値、テール選好次第で採用可(ens_lab に gate 実装済み)。

## 4. 【ブレイクスルー】金(XAUUSD)のトレンドフォローは本物 — セッション初の同一テール改善

FXで全滅したトレンドフォローを、新規取得した **XAUUSD 10年 M1**(Dukascopy)で追試したところ、
**時系列モメンタム tsmom lb60 D1 が単体で成立**(n=150, プールPF 2.12, IS 1.75 / OOS 2.45,
+72bps/トレード, コスト$1.20(3倍)でも PF 2.0)。「FXは平均回帰・金はトレンド」の教科書的構図が
コスト後でも確認された。なお同じ金で**平均回帰v2構造は不成立**(PF 0.87、ショートが2024-25大相場に
轢かれる)=資産ごとにエッジの向きが逆。

**チャンピアンとの統合(同一テール・プロトコル)— 敵対検証5項目すべて通過:**

| 構成 | empirical CAGR | robust(p95=20%) CAGR |
|---|---|---|
| チャンピオン mp11(基準) | +23.83% | +15.42% |
| **champ + 金tsmom(w=0.5, 枠1)** | **+28.88%** | **+17.37%(+1.95pp)** |

- 月次相関 +0.11(ほぼ直交)。較正ノイズ帯(±0.6pp)を明確に超える、**セッション初の真の前進**。
- w高原: w∈[0.25,0.65] 全て+16.7%以上(滑らか)。lb高原: 40/60/80 全て基準超(lb40 +18.1%)。
- IS較正→OOS素検証: OOS CAGR +32.5%→**+40.4%**。
- **金マニア期(2024-26)除外でも +13.2%→+15.0%** = 大相場1本依存ではない。
- seed感度: p95 = 0.201〜0.204 で安定。
- 正直な注意: 金tsmomのISエッジは薄め(1バー遅延で IS PF 1.06)= エッジの厚みは2022以降に偏る。
  小さい枠(budget=1, w≈0.5)で「保険つきの追い風」として持つのが適正で、増し盛りは不可
  (w=2.0 で robust +10.1%に劣化)。

## 5. 結論

**FXメジャー+クロス(2016-2026, コスト後)では、トレンドフォローはどの族・どの時間足・どの
パラメータでも単体エッジが存在しない**(67構成生存ゼロ)。押し目D1の PF1.06 も IS 赤字。
一方、**別資産=金では tsmom が本物**で、チャンピオンに小枠で統合すると robust CAGR +15.4%→+17.4%
(+13%相対)の**正当な改善**になる。これは [10][11] が指摘した「フロンティアを動かすのは別資産だけ」
の最初の実証でもある。
