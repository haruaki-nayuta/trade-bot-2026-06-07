# レポート02: 磨き込み・別系統手法・実現可能性の確定

日付: 2026-06-07 / 前報: [01](01_baseline_landscape.md)

## 1. やったこと(イテレーション2–3)

レポート01の最有望(平均回帰)を起点に、PF と「毎年プラス」を**カーブフィット無しで**底上げできるか、
複数の正当な手段を網羅検証した。

| 施策 | 仮説 | 結果 |
|------|------|------|
| レンジ・レジームフィルタ `meanrev_range`(ADX<th時のみ平均回帰) | トレンド期の負けを除外しPF↑ | プラス年率64%・PF1.05 で**ほぼ不変** |
| トレーリング/利確(TSL2%, SL+TP) | 損小利大でPF↑ | PF中央値1.05→1.15に微増、プラス年率は55%に低下 |
| リスク%サイジング(1%) | 年次安定↑ | ほぼ不変 |
| 合成ポート(平均回帰+モメンタム) | 非相関で毎年プラス | プラス年率55%・PF1.00。**改善せず** |
| 合成ポート(平均回帰+モメンタム+ブレイク) | 同上 | プラス年率55%・PF1.01。**改善せず** |
| セッション・ブレイクアウト `session_breakout`(アジアレンジ放れ) | 時間帯構造のエッジ | 2/7ペアのみプラス・PF0.96。出口調整も無効 |
| クロスセクション・モメンタム(7ペア相対強弱L/S) `exp03.py` | 資産横断の頑健エッジ | **通算マイナス**(-31〜-139%)・プラス年率27-36%。USD主導で分散出ず |

## 2. 決定的な所見

### (a) 合成しても「悪い年」が共通 → 分散で救えない
平均回帰・モメンタム・ブレイクの**いずれも 2019・2020・2025 が負け**。エッジが特定レジーム(低ボラ/急変)で
同時に消えるため、ペア分散・手法分散をしても年次のマイナスが残る。「毎年プラス」の本質的な壁。

### (b) PF2.0 は「少数の選り抜きトレード」でしか出ず、OOSで崩れる(=カーブフィットの罠)
`meanrev_range` を H4 でグリッド探索すると EURUSD で **PF 2.38(Sharpe0.93, 勝率68%, DD-3.3%)** が出る。
だが:
- **取引は10年で81回=年8回**(目標の年100に遠く及ばない)。
- **ウォークフォワードではOOSプラスが40%のみ**。この高PFパラメータは後半foldでしか現れず、
  前半foldは別パラメータを選んで-10.8%/Sharpe-1.66と崩壊。
→ 高PFは「取引を絞り込み、過去に都合よく合わせた」結果であり、**前向きには再現しない**。これは禁止された過剰最適化そのもの。

### (c) 多取引・頑健寄りの素の実力
`rsi_meanrev`(H4, 年373取引/ポート)でも、ウォークフォワード平均 Sharpe -0.38・平均リターン-2.95%。
EURUSD単体OOSは Sharpe+0.63/PF1.14/DD-9%と**正の期待値はある**が、全ペア・全年では崩れる。

## 3. 実現可能性の確定的結論

**「年100取引/ペア・PF2.0以上・どの年もどのペアもプラス」は、カーブフィット禁止の制約下、メジャー7ペアでは到達不能。**
根拠:
- トレンド/平均回帰/ボラ放れ/セッション/クロスセクション/合成/各種出口・サイジング を H1/H4/D1・7ペア・11年で網羅検証。
- 最良でもポートフォリオ PF ≈ 1.0–1.15、プラス年率 55–64%。
- PF2.0 が出るのは取引数を極端に絞った時のみで、ウォークフォワードで非頑健と露呈。
- メジャーFXは世界有数の効率市場で、プロのシステム系FXでも通常PF1.2–1.4・Sharpe~1。PF2.0常時は構造的に困難。

## 4. 到達可能な「最良システム」(正直な推奨)

| 項目 | 内容 |
|------|------|
| 中核 | 平均回帰 `rsi_meanrev`(H4)を主軸。レンジ局面で機能 |
| 現実的指標 | EURUSD OOS Sharpe ~0.6 / PF ~1.14 / 最大DD ~9%(浅い)/ 年300+取引 |
| 弱点 | JPY系・一部コモディティ通貨では負け越し。低ボラ/急変年(2019,2020,2025)で停滞 |
| 改善余地 | ①機能するペアに限定 ②ボラ・トレンドのレジーム検出で停止 ③小ロット・分散運用 |

## 5. 次アクション(ユーザー判断が要る分岐)

目標が現状の数値定義のままだと到達不能のため、**受入基準の現実的な再設定**を提案する(下記いずれか):
- **(A) 指標を現実水準へ**: 例)ポートフォリオ年次プラス率 ≥ 80%、PF ≥ 1.3、最大DD ≤ 15%、年100取引(ポート合算)。これなら磨き込みで狙える。
- **(B) ペアを限定**: 構造的に機能する EUR/USD・GBP/USD 等に絞り「そのペアで毎年プラス」を目指す。
- **(C) 別データ前提**: カバー金利(キャリー)・経済指標・板情報など、価格以外のエッジ源を導入(要データ拡張)。
- **(D) 探索継続**: 上記制約のまま、さらに別アイデア(マルチTF合流・適応的レジームスイッチ等)を試す(PF2.0常時は引き続き困難な見込み)。
