# 23. サブバー遅延用量曲線(d1 の1時間刻み) — d1=4h は曲線の肩。細分化に果実なし

日付: 2026-06-12 / 対象: `confluence_meanrev_v2_d1` + P4.0 mp8 / exp67

**問い(ユーザー)**: d1(エントリー1バー遅延)をもっと細かく変動させるとどうか。

**結論(先出し)**: **採用なし。d1(=シグナル close から 4h)が「ゲートを通る最大用量」のまま**。
1時間刻みの測定で2つの新知見を確定:
(a) **4h 未満の早出しは単調劣化**(逆行第1波の回避便益は最初の1バーかけて貯まる)、
(b) **4h 超の見かけの利得(+6h:+1.3pp/+8h:+1.8pp)は禁止済み d2 トラップの早期発現**
(プール利得の100%超が2020年単年。ロールオーバー除染でも消えない=汚染でなくレジーム依存)。

---

## 0. 設計

シグナル(0h位相 H4 グリッド)は固定し、**執行時刻だけ**を X∈{0,1,2,3,4,5,6,8}h で刻む
(exp46 で死んだ位相オフセット=シグナル自体の別グリッド生成とは別物)。約定は自銘柄 M1 close
(クロスは脚 M1 inner-join 合成)、休場時は次 tick。因果ゲート: X∈[4,8) は d1 と同一
(t+1 close で exit 未発火)、X<4 は将来 z を知り得ないため全件保有。
検算アンカー: X=0 が d0(sum 差 +0.008=M1/H4 丸め差)、X=4 が d1(n=1207 一致, sum 差 +0.010)を再現。

## 1. 用量曲線(口座 robust seed0, 対 d1)

| X | rob_s0 | 対d1 | プール段の素性 |
|---|---|---|---|
| 0h(=d0) | +16.61% | -1.82pp | アンカー(d0 +16.4%再現) |
| 1h | +17.45% | -0.98pp | 単調回復の途中 |
| 2h | +18.12% | -0.31pp | 〃 |
| 3h | +18.30% | -0.13pp | 〃 |
| **4h(=d1)** | **+18.43%** | 0 | 採用構成 |
| 5h | +18.02% | -0.41pp | 約定の33%がロールオーバー窓 |
| 6h | +19.69% | +1.26pp | **最良年2020除外でプール差マイナス(-0.017)** |
| 8h(=d2) | +20.22% | +1.79pp | 〃(-0.004)= reports/18 の d2 トラップ再現 |

- **X<4h**: -1.82 → -0.98 → -0.31 → -0.13 → 0 と滑らかに単調。早く入るほど第1波を
  食らう量が比例的に増える=便益は1バーかけて連続的に貯まる。サブバー早出しに最適点なし。
- **X=6h の +1.26pp の解剖**: 約定33%が UTC20-23 窓に落ちるため除染変種
  (20-23時の約定を23時以降へ送る)を実測 → **+19.99%とむしろ微増=汚染は主因でない**。
  しかし除染後もプール利得 +0.106 のうち **2020年除外で -0.007(利得の107%が2020単年)**
  = d2(2020年88%集中で禁止)と同一のレジーム依存が **+6h から既に始まっている**。
  2020=COVID 年の深い多バー逆行では「長く待つほど得」だったという単一レジーム現象であり、
  構造的エッジではない(IS/OOS分解も IS+0.081/OOS+0.016 と IS偏重)。
- X=5h の謎の凹み(-0.41pp)は曲線が4h超で単調ですらないことを示す=隠れた最適点なし。

## 2. 結論

d1 の「次バー close まで待つ」は、(a) これ以上早いと便益を比例的に手放し、
(b) これ以上遅いと2020年単年のレジーム依存(禁止済み d2 の顔)を拾い始める、
**ちょうど肩の位置**にあると1時間解像度で確定した。執行タイミング軸は
d0/d1/d2/d3(exp47)・位相(exp46)・週末(exp58)・サブバー(本実験)で完全に掃引済み=恒久閉鎖。
