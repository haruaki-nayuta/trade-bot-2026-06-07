# 11. トレンドフォロー総合評価 — 金の押し目買いが唯一の生存者

- 日付: 2026-06-11
- 課題(ユーザー指定): トレンドフォロー手法を評価。**IS・OOS 両方で PF>1.0 必須・年間24取引以上・
  利益はなるべく大きく(チャンピオン手法以上)**。発想は自由。
- 手法: Workflow 2段(候補8系統×並列実装+IS/OOS検証 18エージェント → 上位3候補×3レンズ敵対検証+
  合成検討 10エージェント)。IS=前70%でのみ最適化、OOS=後30%は素検証(fxlab.evaluate と同一プロトコル)。

## 敵対検証の詳細(上位3候補 × 3レンズ)

### pullback_trend_gold — 生存(overfit: concern / regime: concern / cost_tail: pass)
- **過剰最適化**: 拡張216組合せの全てで Sharpe>0(99%が PF>1.3)、採用点は full 2位/216 で孤立ピーク
  なし・グリッド端なし。dipフィルタを完全無効化しても Sharpe0.95 = 構造が効いており1点の偶然でない。
  WF 5/5 プラス・パラメータ不動。**concern の中身**: 報告されていた TSL2% ヘッドライン(+156.5%)は
  全期間知識を使った選択で IS-Sharpe 規律では SL2%+TP4% が選ばれる(full +89.7%/PF1.79/DD-8.7%)。
  リターン最大化なら**オーバーレイ無しの素(+130.9%)が正解**(docstring を修正済み)。
- **レジーム依存**: 利益の~78%は2023年以降。ただし暴騰前2016-2022単独でも **+30.0%/PF1.56/
  Sharpe0.68/DD-10.5%/年27.8取引** で明確に正、しかも金B&Hがマイナスの2021-22に PF3.7/2.9 =
  **横ばい相場での真のタイミングα**を持つ。ロング専用化は暴騰フィットでなく構造的に正当
  (対称ショートは暴騰前 PF0.86 / 暴騰期 PF0.71 と両レジームで負け)。
- **コスト・テール**: スプレッド2倍+手数料の複合最悪でも IS PF1.43 / OOS PF2.52 / full +114%。
  **1バー遅延で劣化なし**(IS はむしろ改善 = バー際のタイミングに依存しない)。risk1% サイジングで
  CAGR 6.1%/DD-10.5%(OOS は CAGR 14.6%/DD-5.9%)。週末ギャップは正のスキュー(安全資産の
  上ギャップがロングに有利、最悪-0.87%)。バー安値ベースの DD 監査も -14.7% vs -13.9% で偽装なし。

### tsmom_ens_gold — fail
- コスト2倍・risk1% は生存するが、**約定1日遅延で IS PF0.94 に崩壊**(IS のエッジが「シグナル翌日の
  即時継続」に全集中)。終値→翌始値約定なら IS PF1.34 で生きるため、運用するなら執行条件が厳格。
- lbセット選択フィットの署名: 採用セットのみ IS PF1.36、近傍セットは IS PF≈0.97-0.99。一方ランダム
  5本セット25個が25/25で full/OOS PF>1 = 効いているのは特定セットでなく金β。
- リターン・Sharpe・DD とも金 B&H に劣後。long-only 化(+162%/PF2.34)が両建てを全指標で上回る。

### tsmom_asym_gold — fail
- 公正な祖先=**ロング専用 tsmom lb360** が full +170.8%/PF3.79/Sharpe0.96/DD-23.1%、OOS PF7.43 で
  本戦略(5パラメータ)を両窓・全指標で支配。docstring の「全指標で上回る」は対称版とのストローマン比較。
- 新規部品が実測で負: ブリッジロング単体 PF0.60(-8.9%)、除去で全指標改善。ショートは10年+4.0%の死荷重。
- OOS PF3.06 はトップ1取引が OOS 利益の49%という集中の産物(トップ5除外で PF0.58)。

### 合成(3戦略ポートフォリオ)— fail = 単独推奨
- 日次リターン相関 0.62-0.71(活動日のみ 0.75-0.87)= 同一資産・同方向の変種で分散効果なし。
- 等ウェイト合成は最良単独の totret を構造的に超えられず、DD-matched レバ較正では「empirical CAGR↑+
  boot p95 悪化」の**レバ偽装署名**を検出(同一テールプロトコル準拠で棄却)。
- reports/10「補完エッジは混ぜても割に合わない」と同型の結論。

## 正直な限界(誇張しないための明記)

1. **これは金の長期強気構造に乗る戦略**。OOS窓(2023-06〜2026-06)は金暴騰窓と一致するため
   OOS PF2.63 は頑健性指標として過大。前向き期待は WF 平均(Sharpe≈0.94・年率7-12%)をアンカーに。
   金のレジームが崩れた場合の実力は暴騰前実測の「年率~4%・PF1.56」水準まで低下しうる。
2. **FX には一切移植不可**(7ペア横断全敗)。「FXトレンドは死んでいる」は今回も再確認(ERゲートでも
   クロスセクショナルでも不成立)。トレンドフォローをやるなら資産は金(等のトレンド資産)一択。
3. チャンピオンとの比較は「単一戦略・同一ハーネス」基準では圧勝だが、**本番資金管理込みの
   チャンピオン v2 ポートフォリオ(z-power, mp11: empirical +23.8%/robust +15.3%)とは土俵が別**。
   risk1% の pullback_trend_gold は CAGR 6.1%(DD-10.5%)であり、口座レベルでチャンピオン体制を
   置き換えるものではない(過去検証で金tsmom小枠統合が robust +15.4→+17.4% だった事実とも整合的で、
   「金トレンドはチャンピオンの補完小枠」が依然として最有力の使い方)。
4. 勝率は ~26% の損小利大型。心理的には「10連敗しつつ年数回のトレンドレッグで稼ぐ」形。
