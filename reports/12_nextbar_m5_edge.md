# 12. 短期足(M1/M5)の「直近100本→次足」統計エッジ探索

**日付**: 2026-06-11
**問い**: M1/M5の直近100本の価格変動から次の足のエッジを統計的に見つけ、利益化できるか。
**答え**: エッジは実在する(全て「反転」方向、M5のみ)が、ロールオーバー・アーティファクトを除染すると**EURUSD限定・年75〜150pips級の薄いエッジ**しか残らない。M1は次足ベースでは構造的に不成立。副産物として、**短期足検証全体を汚染する重大な計測アーティファクト**を特定した(§3)。

## 1. 方法

- 共通プロトコル `research/lab/nextbar_common.py`: ターゲット=次足close-to-close(pips)、train(2016-06〜2022-12)/test(2023-01〜2026-06)分割、極値ビン閾値はtrain分位のみ、Spearman IC+条件付き平均pips+t値+ホライズン減衰(h1〜h20)。
- フェーズ1: 7特徴量ファミリー並列スキャン(モメンタム/ボラ状態/ローソク解剖/セッション/レンジ位置/出来高/パターンマイニング)。Workflow 7エージェント。
- フェーズ2: 敵対的検証5並列(クロスペア再現×2 / リーク・再実装監査 / 合流の限界価値と出口設計 / 週末ギャップ)。
- 再現: `research/experiments/nb_*.py`(各ファミリー+検証)。

## 2. ベースラインの現実(コストの壁)

M1は完全予知ですら平均0.3p/バーしか残らない。**全ファミリーがM1で「構造は再現するが単価1/5〜1/20」を確認 → M1は次足トレード対象として閉鎖**。

## 3. 最重要の副産物: ロールオーバーBIDアーティファクト(全短期足検証の罠)

UTC 20-22時(NY5時ロールオーバー)はスプレッドが2-5pipsに拡大し、BID建てDukascopyデータではbidが機械的に沈む→正常化が「V字反発」に見える。**独立した複数エージェントが同一の罠を別経路で発見**:

- h21のロング+1.49p(t=8.1)等、フルサンプル最強級の「エッジ」が全てこれの変装(閑散ドリフト・効率リバーサル・ギャップ埋め・クライマックス反転の一部)
- z20平均回帰**ロング側エッジの約4割**がこの窓由来。ショート側はほぼ無汚染(bidが沈む方向の非対称が署名)
- クロスペアでは汚染がさらに激しい: ロング側チャンピオンの見かけの好成績はスプレッドの広いペアほど巨大(USDCAD +2.67p/USDCHF +2.27p)= スプレッド拡大幅に比例 → **除外後にコスト超えするペアはゼロ(EURUSDのみ生存)**
- 週末ギャップ埋め(test h20 +11p, t=3.5)も崩壊: ギャップ無しの週でも日曜オープンh20 +5.8p(t=15.7)、7ペア・USD両方向が同時プラス=mid価格の現象として成立不能
- M1の分単位時刻統計は2023年以前のフィードアーティファクト(交互sawtooth t=-11がtestで消滅)を含む

**確立した規律**: ①短期足(M15以下)の統計は「フル」と「UTC20-23エントリー除外」の両方を計測し除外後を判定主軸にする ②「全ペア・USD両方向が同時に同符号」はアーティファクト署名 ③執行ルールとして20:50-21:05に成行を置かない。→ memory `rollover-bid-artifact-m5` に保存済み。

## 4. スキャンの主な発見(EURUSD M5, test実測)

- **次足エッジは全て反転**。継続/ブレイク追随は26特徴量×全分位で全滅(100本新高値クローズの次足は-0.10p=逆行)
- **非対称構造**: 買いは短窓z10が最強、売りは中窓z50/3本速度ret3がh3-h5ピーク
- **ER仮説の逆転**: 反転はレンジ(低ER)でなく「高ER=ほぼ一直線の効率的な急落」の直後に最強(z10極値: 低ER +0.00p vs 高ER +0.54p)
- **ボラゲート**: z20極値の平均回帰は高ボラ・高出来高でのみ働く(低ボラ・中出来高の極値は死んでいる)
- **教科書パターンは全滅**: 包み足・インサイドバー・ハンマー(長い下ヒゲ)は無情報。反発を予測するのは長い下ヒゲでなく「ヒゲなしの安値ベタ引け」=最大ストレッチ
- パターンマイニング: 生き残りは連騰連落リバーサルのみ。交互チョップ・三段下げ・V字はtrainの幻影(多重検定の罠の実例)

## 5. 敵対的検証の結果

- **リーク・バグなし**: 独立再実装でチャンピオン+0.537p/t=4.96を正確に再現。特徴量にshift(-)なし、train分位規律も健全。ただし発見時の定義記述に不備(「z10のq02」は実際にはhiER限定後の条件付き分位)
- **執行現実性**: close約定はt+1 open比+0.07p楽観的。現実的グロスは+0.47p(除外後)→**次足単発ではコスト0.6p未満**
- **h20まで伸びる」は約25%がアーティファクト**: 除外後h20 +0.71p(t=1.5)で有意性喪失
- **出口の罠**: zクロス出口はtrainで最良/testで最悪と完全逆転=train専用の幻。時間出口(ロングN=1、ショートN=3-5)だけが両期間で頑健
- **train-argmax→test ≈ ゼロ**: 64組合せ×6出口をtrainだけで選んでtestに当てると日次+0.01〜0.05p。出口・組合せ選択にtest情報が混入しうることへの警告

## 7. 総合判定

**非推奨(単独運用には値しない)、ただし本物の微小エッジとして記録価値あり。**

- 統計的実在: ✅(除外後もロングt=3.8、ショートt=3-4、OOS全年プラス)
- 経済性: ⚠️ 誠実な期待値は**合算+0.2〜0.4p/日(年75〜150pips)**。OOSのPF1.42は2022年以降の高ボラレジーム+出口選択のtest寄りバイアスを含む上限値。IS期はほぼ損益分岐であり、低ボラレジーム回帰なら消える
- ノイズ確率3〜4割(多重検定込み)。実弾前にデモでフォワード3ヶ月が必須
- 復活条件: 往復コスト<0.4pの口座(リミット注文でのスプレッド片道化含む)なら土俵が変わる

**チャンピオン(confluence_meanrev_v2 H4)との関係**: 完全に別の時間スケールの現象であり、ポートフォリオ追加は§iron-triangleの「混ぜても割に合わない」既知パターンの再来になる可能性が高い。追加検証するなら失血窓での条件付き貢献(reports/10の方法)で。
