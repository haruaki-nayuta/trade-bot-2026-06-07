# 16. 改善レバー検証ラウンド — 期待値上位5本を同一テールプロトコルで判定: 全5本 reject

日付: 2026-06-11
前提: reports/15(利益の解剖)が列挙した有望レバーのうち、オフラインで検証可能な期待値上位5本を検証。
方法: Workflow 5エージェント並列。各レバーに **tail_protocol フル適用**(ペアシード0-4でrobust/empirical両較正・IS-argmax監査・2022除外・レバ偽装署名・全年プラス維持)を義務付け。ベースラインは全エージェントが同一スクリプト・同一シードで再計算し、参照値(mp8 robust≈+16.3%/emp≈+24.6%)の再現を集計規約の検算とした(全5本で再現一致)。
再現: `research/experiments/exp41_volwin.py` 〜 `exp45_limit_entry.py`(実行コマンドは各ファイル先頭)。

**結論: 全5レバー reject。現行本番構成(v2・19銘柄・f(z) P=4.0・mp8)は、検証した5次元すべてで局所最適と確定。** ベースライン robust(p95=20%) 5シード平均 = **+16.41%** / empirical 20% = **+24.64%** が現在の到達点。

## 2. 各レバーの要点(なぜ死んだか=機構)

### ① vol_sd_win(exp41)
- w25単体は robust +19.22%(+2.82pp・5シード全勝)と魅力的に見えるが、細断面 22/24/26/28 を測ると **w25だけが浮く**(rob_s0: w22 +15.37 / w24 +16.37 / **w25 +19.24** / w26 +17.77 / w28 +12.12)。プールレベルでは w24/w26 とも基準割れ。H4窓長±1本で消える改善は構造ではない。
- さらに emp較正で p95 が約1.2pp悪化(シード横断で再現)=レバ偽装署名。**高原ゲートとテールゲートの二重抵触で reject**。
- 収穫: reports/15 gap_2 の「単調増の端逃げ」懸念は否定(w35/40は−4.5pp級の大幅劣後)。**「25が偶然」と確定し、この軸は閉鎖**。

### ② f(z)×g(vol)(exp42)
- 核心の発見: reports/15 の「vol Q4が純益47.5%」勾配は **full-sample四分位(事後分位)に基づく記述的事実**であり、**因果rolling(1000)ランクに置換すると spearman +0.09 まで希釈・非単調化**(IS Q2 PF0.91、OOSは中位が最良と形が入れ替わる)。つまり勾配の大半は「後から見ればあの時代は高ボラだった」というレジーム知識で、**エントリー時点では知り得ない=サイジングに変換不能**。
- 全12構成が robust でベースライン未満。唯一 empirical が伸びる二値Q1抑制は p95 −29.9% のレバ偽装署名+2022が改善の67%。
- 成分分解(低vol抑制/高vol増し)も両方 +0.1〜0.2pp 止まりで p95 悪化 → **排他候補だった「低ボラQ1抑制フィルタ」(reports/15 §2-3)も同根拠で閉鎖**。

### ③ 25銘柄化(exp43)
- プールレベルの+7.4%は再現したが、**mm層で完全逆転**: 新6クロス(CHF/NZD脚)は既存19と**相関した深いMtM含み損クラスタ**を共有し、emp較正p95を−27.8→−36.1%に肥大させ、較正kを8.27→6.94に強制低下。同一k固定でも新6の寄与は−0.91pp(較正以前に純減)。2018年マイナス転落で全年プラスも崩壊。11年中9年で悪化=単一年依存ですらない。
- IS選択も19銘柄(IS robust +14.46% vs +8.45%)=後知恵の問題なし。**プール集計と口座テール会計は別物**という reports/10(補完エッジ全滅)と同型の教訓。
- GBPNZD単独はプールでは本物に見える(PF1.67・10/11年プラス・6pips耐性)が、6本中1本の事後選別は規律違反のため不採用。

### ④ mp11昇格＝M1粒度監査(exp44)
- H4粒度の優位+0.99pp(5シード全勝・IS-argmaxもmp11・OOSも優位)は再確認。**棄却理由は専らM1粒度ゲート**。
- M1リプレイ(372.6万本グリッド、クロスは脚M1 inner-join合成): mp8 の真の谷は H4比 **1.036**(ほぼ正確)に対し、mp11 は **1.161**(谷=2022-03-07 01:51、ウクライナ侵攻直後の月曜早朝)。機構: mp11の追加3玉は**ボラクラスタでちょうど増える**ため intrabar テールを構造的に深くする。
- 掛け目込み実効CAGR(M1 DDを自構成のH4較正DDに一致させるまでk反復): **mp8 k×0.965 → +15.80%** / mp11 k×0.861 → +14.85% で**優位が逆転**。代替規約(同一絶対M1 DD)でも mp11 同着止まり。
- 証拠金は両構成とも最大総レバ9.4x ≪ 25x で問題なし。
- **副産物(恒久化)**: 本番19銘柄構成のM1監査が完了し、**mp8の掛け目は k×0.965**(exp24のメジャーのみ×0.95より精密)。p95_M1近似: mp8 −20.7%。

### ⑤ エントリー指値化(exp45)
- fill率91.8%・約定ゲイン+6.06%(理論上限+6.78%の89%)まで回収できるのに、**未約定8.2%(n=99)の成行ret平均+31.1bps・勝率77%** vs 約定コホート+14.4bps=「刺さらないのは即走った勝ち」の逆選択を直接実測。次バー成行フォールバックの遅延損−8.11%が回収を上回り、プール純益−2.69%。
- 規約感度に頑健: BID規約−2.69%/mid解釈−4.14%/最楽観のタッチ約定でも+1.86%止まり(robust換算+0.5pp未満かつ非現実的)。見送り型(b)は−10.08%でさらに悪い。20:00窓を厳格化すると悪化方向。
- reports/15 の「1バー遅延+2.7%」は**平均トレード**の性質であり、指値が遅延させるのは逆選択された未約定コホートのみ=前提の読み違いを実測が正した。

## 3. 帳尻と含意

1. **reports/15 §2-2 の有望レバーのうち5本(①②③④⑤+派生の低volフィルタ)を閉鎖**。これで収益レバーの残存は: スワップ・マークアップ実測/ブローカー選定(外部データ)、デモフォワード(外部)、金tsmom×P4合成(reports/11のコードがこのブランチに無く要再実装)のみ。**バックテスト内で取れる果実は尽きた可能性が高い**。
2. reports/15 への訂正: 「vol Q4=純益47.5%」は記述的には正しいが**因果シグナルではない**(事後分位)。「失血署名=平均ER上昇のモニタ」も同じ目で再吟味が必要(モニタ用途なら事後で可、配分に使うなら因果検証必須)。
3. プロトコルの有効性が改めて実証された: 5本中4本は「単純な検証」なら adopt されていた(①は5シード全勝+2.82pp、④はH4で全ゲート通過)。**細断面・M1粒度・因果ランク化・逆選択の直接測定**が幻影を切った。
4. 本番運用への反映: `mm_production.py` の掛け目は **k×0.965**(M1監査実測)に更新可能。mp11オプションは「H4上は優位だがM1粒度で逆転」と注記すべき。
