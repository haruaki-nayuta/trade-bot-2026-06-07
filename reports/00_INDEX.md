## 環境に追加した検証ツール(このセッション)
- `fxlab/yearly.py` … 決済年でトレードをグループ化し **ペア×年** のPF/損益/取引数を集計。`portfolio_yearly`(7ペア等加重合算)・`acceptance`(目標合否判定)。
- `yearly.py`(CLI) … `uv run python yearly.py <戦略> --tf H4` で年次マトリクス+目標判定。
- `screen.py`(CLI) … `uv run python screen.py --tfs H4 D1` で全戦略を**目標基準**で一括ランク。
- `tests/test_yearly.py` … 年次集計の正しさ(年次PnL合算=全期間PnL 等)を担保。
