# 13. 短期足トレンド検証 + 方法論の敵対的デバイアス・レビュー

日付: 2026-06-11 / 前報: [12](12_trendfollowing_sweep.md) /
動機: ユーザー指摘「①逆にもっと短い時間足では? ②既存手法を正当化するためにエッジを過小評価していないか敵対的にレビューせよ」

**結論(先出し)**:
①短期足(M15/M30/H1)のトレンドフォローは **グロス(コスト0)でも全構成マイナス** = コストに殺されているのではなく、シグナル自体が逆向き。唯一の例外はロンドン序盤の極小ドリフト(+0.1〜0.4bps)でコストの1/5〜1/10。
②**実装バグはゼロ・コスト計上も正確**(手書き再計算と1e-15一致)。しかし**フレーミングに3つの実在する反トレンド・バイアス**を発見: (a)USD相殺ユニバース(FX19銘柄の年次ドリフト|t|>2が0/209件=トレンド不在の銘柄群で裁いていた) (b)スワップ無視(JPYロングのトレンド損益はスワップ込みでほぼ倍増) (c)集計が**ロング側のポケット**(XAUUSDロング・USDJPYロング、2族×IS/OOS一貫)を平均に埋没させていた。
→「**この検証環境(FX中心・USD相殺・2016-26・価格のみ)ではトレンドは効かない」は正確だが、「トレンドは効かない」への一般化は過大**だった。是正後の収穫が[12]の金tsmomであり、本レポートで金ドンチャンロング(robust判定)とUSDJPYロングの統合判定を追加。

---

## 1. 短期足トレンド(M15/M30/H1)— グロス診断つき全滅

| 族 | 構成数 | net | gross(コスト0) |
|---|---|---|---|
| ドンチャン(H1/M30/M15) | 9 | 全滅(PF 0.81前後) | **全滅**(−1.5〜−5.5bps/件) |
| tsmom+ヒステリシス(1日〜15日相当) | 18 | 全滅(PF 0.71-0.85) | **全滅**(−0.5〜−5.4bps) |
| セッションORB(LDN/NY, M15/M30) | 8 | 全滅(PF 0.77-0.87) | LDNのみ極小プラス(+0.1〜+0.4bps)、NYは負 |

- コストの寄与は一律 ≈2.1bps/件。**コストゼロでも負ける**=「短い時間足ほどFXは平均回帰的」が定量で確定。
- ヒステリシス(不感帯)は取引数を50-75%削減しネットPFを改善するが、グロスの per-trade エッジはむしろ悪化=**コスト削減装置であってエッジ生成装置ではない**。
- ロンドンORBの正ドリフトは文献どおり実在するが、リテールコスト(1-3bps)の1/5〜1/10で取れない。
- 注意: 別セッションの知見([[rollover-bid-artifact-m5]])のとおり M15 以下は UTC20-22 のBIDアーティファクト混入リスクがあるが、本結果は全構成マイナスなので結論に影響しない(偽エッジはロング方向に出る)。

## 2. デバイアス・レンズ1: 非対称出口(トレーリングストップ)

「反対シグナルでドテン」でなく tsl_stop {1,2,4%} で利を伸ばす本来のTF出口に変えても:
- FX19: 全18構成 PF<1 のまま(tsl=1%で数pt改善する構成はあるが損益分岐に遠い)。D1で広いtslは悪化。
- XAUUSD tsmom D1: tsl=1%で PF 1.99 だが**総リターンほぼ半減**(+108%→+58%)。
  [[champion-nostop-optimal]] と同じ「無ストップがPareto最適」構図が金トレンドでも再現。
- 方法上の発見: vectorbt from_signals は反対エントリーで自動ドテンするため exits=反対シグナルは冗長
  (エントリーのみ版と数値完全一致を確認)。

## 3. デバイアス・レンズ2: M1ストップ注文エントリー(本物のタートル式)

「終値確認→終値約定」はブレイクの行き過ぎを払う分トレンドに不利、という仮説を M1 実データで検証
(メジャー7+XAUUSD、レベルは前バー確定値、ギャップは寄り約定、片側半スプレッド):
- **改善は実在する**: 速い構成で +3.7〜+6.9bps/件(H4 55x20: PF 0.853→0.921)。
- **しかし FX は 8構成中8構成とも PF<1 のまま**(損益分岐に必要な+7〜40bpsに届かない。
  タッチ式は終値までに失速する whipsaw も拾うため約35%トレードが増え、改善を一部相殺)。
- XAUUSD は両執行方式でプラス(stop H4 55x20: PF1.23/OOS1.38/n=393)=金のエッジは執行方式非依存。

## 4. デバイアス・レンズ3: クリティック監査(コード・集計・コスト)

- **フリップ会計: バグなし**。tsmom D1 EURUSD を vectorbt 抜きの手書きループで再計算し
  n・sum_ret・全トレードが一致(差<1e-15)。ドテンの二重計上・取りこぼしなし。
- **コスト: 正確**。往復ちょうど1フルスプレッド(0.600pips を数値確認)。過大課金なし。
- **ボラ正規化(ATRリスク均等)重み付け: 符号不変**。等加重がボラ加重の勝者を隠している事実はない。
- **集計マスキング: 実在**。(i)「全滅」の主犯はショート側(tsmom ロングは OOS PF 0.997≒トントン)。
  (ii) 一貫プラスのポケット: **XAUUSDロング**(tsmom PF4.15 / donch PF1.78 n=154 IS1.26/OOS2.45)と
  **USDJPYロング**(tsmom PF2.01 n=78 / donch PF1.33 n=161 IS1.38/OOS1.29)— 独立2族×IS/OOS両方。
  (iii) 2022年は USDJPY PF23・GBPUSD 6.6 等の大勝ちが USD相殺クロスに希釈されプール計マイナス。
- **プロトコル外バイアス(影響度順)**:
  1. **ユニバース選択(大)**: 12/20がUSD相殺合成クロス。FX19の年次ドリフト|t|>2 = **0/209件**
     (XAUUSDは9.1%)。ユニバース自体が平均回帰チャンピオンの安定化のために選ばれた経緯があり、
     「トレンドをチャンピオンのホームグラウンドで裁いていた」。
  2. **スワップ無視(中〜大)**: tsmom JPYペアロング2022+ は価格+0.487 に対しスワップ+0.434
     (+39bps/件)で**ほぼ倍増**。長保有のトレンドに効き、短保有の平均回帰にはほぼ効かない=非対称。
  3. 同バー終値約定(小・むしろ微不利): 翌バー寄りに変えると tsmom EURUSD は+1.3bps/件改善。
  4. 期間2016-26(中): FXの大トレンド期(2008, 2014-15)を含まない。

## 5. ポケットの統合判定(同一テール・プロトコル)— トレンド・スリーブ確定

クリティックが発掘した2ポケットを、[11]のプロトコル(empirical/robust 両較正+敵対検証)で統合判定:

| 構成 | empirical CAGR | robust(p95=20%) CAGR |
|---|---|---|
| チャンピオン mp11(基準) | +23.78% | +15.40% |
| + 金tsmom(w0.5, 枠1)[[12]] | +28.88% | +17.37% |
| **+ 金tsmom + USDJPYロングdonch(w0.5, 枠1)** | **+32.53%** | **+18.56%(+21%相対)** |
| (USDJPYレッグを tsmom D1 に替えても) | +30.57% | +18.60%(=族非依存) |

USDJPYロング・レッグの敵対検証(すべて通過):
- **1バー遅延**: PF 1.33→1.27(IS 1.35 / OOS 1.22)で生存 — 金tsmomと違い執行遅延に頑健。
- **w高原**: w∈{0.25, 0.5, 0.75} → +18.0/+18.5/+18.8% で滑らか。
- **レジーム依存テスト**: 円安・金マニア期(2024-26)を除外した2016-2023でも
  champ +13.12% → 統合 **+16.45%(+3.3pp)** = 後半の大相場1本に依存していない。
- 金ドンチャンH4ロングも Workflow 検証で robust 判定(遅延・コスト1.5倍・2022除外生存。
  ただし利益の~40%が2025年=金レジームベットの性格は明記)。
- スワップ非計上はこのレッグには**不利方向**(USDJPYロングは2022+でスワップ受取≈+39bps/件)
  =実際の期待値は表より高い側。

**正直な注意**: 統合のempirical p95 は −31.8%(基準−29.0%)とやや深く、empiricalモードでの
運用は[09]のテール注意がやや増す。robust(p95=20%)モードでは同一テールで+3.2ppが確定値。
トレンド・スリーブは「小枠(各1枠, w≤0.5-0.75)」が条件で、増し盛りは劣化する([12])。

## 6. 結論

- 「FXメジャー+クロスの価格リターンには、どの時間足にもトレンドの正エッジがない」は**グロス診断・
  非対称出口・ストップ注文執行・会計監査のすべてを通った後でも正しい**。
- ただしそれを「トレンドフォロー全般の死」と読むのは誤り。**正しい読みは「エッジは資産と方向に偏在する」**:
  XAUUSDロング(及び限定的にUSDJPYロング)はトレンドが実在し、チャンピオンの失血源と逆相関の
  エクスポージャとして統合価値がある([12]の金tsmom = robust +15.4→+17.4%)。
- 検証環境の正直な限界として **スワップ(キャリー)非計上**を明記する。JPY系ロングのトレンド/
  チャンピオンのJPY系ショート塩漬けの双方に影響するため、実弾前にスワップ計上の口座シミュが必要
  ([[live-deployment-broker-landscape]] の宿題と同一)。
