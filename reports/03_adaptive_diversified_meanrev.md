# レポート03: 適応型・分散・ボラ回避の平均回帰 — 到達可能な最良システム

日付: 2026-06-07 / 前報: [02](02_refinement_and_feasibility.md) / 方針: ユーザー選択「制約そのままで探索継続」

レポート02で特定した2大根本原因(**通貨依存** と **レジーム依存=悪い年の共通化**)に、
カーブフィットを避けつつ正面から効く手段を順に積み上げ、年次の安定を大幅に改善した。

## 1. 積み上げた改善(すべて経済合理・非カーブフィット)

| # | 施策 | 効いた根本原因 | 結果(H4, size=value) |
|---|------|---------------|----------------------|
| 1 | **Zスコア正規化** `zscore_meanrev`(固定RSIしきい値→偏差/σで自己正規化) | 通貨依存 | 同一パラメータで **6/7ペア通算プラス**・勝率65% |
| 2 | **二重確認** `confluence_meanrev`(Zスコア × RSI 同時行き過ぎ) | 確信度/ダマシ | **全7ペア通算プラス**・min PF 0.98→1.07 |
| 3 | **クロス分散**(majors から EURGBP/AUDCAD/EURJPY 等13本を合成・計20対象) | レジーム依存 | プラス年率 64%→**82%**(2018・2026 がプラス転換) |
| 4 | **ボラ・レジームフィルタ**(実現ボラ上位=危機局面は建玉停止) | レジーム依存 | プラス年率 82%→**91%**(2020 コロナ年がプラス転換) |

**逆に効かなかったもの(02の確認を補強)**: トレンド/ブレイクのスリーブ追加(レジーム切替)は
プラス年率を 64%→45% に**悪化**。メジャー/クロスFXでは順張りは net マイナスで、平均回帰こそが唯一の頑健エッジ。

## 2. 到達可能な最良システム(現時点のチャンピオン)

**`confluence_meanrev`(Zスコア×RSI×ボラフィルタ)を 20対象(7メジャー+13クロス)に等加重・固定パラメータで適用。**

20対象ポートフォリオ年次(ボラフィルタ vol_pct=0.70, exp06):

| 指標 | 値 |
|------|-----|
| プラス年率 | **91%(10/11年)** ※負けは 2024 のみ |
| PF 中央値 | **1.20** |
| 年間取引(合算) | 約340 |
| 個別の強い対象 | AUDCAD PF1.94 / EURJPY 1.54 / EURUSD(ボラフィルタ時)PF中央2.04 |

メジャー単独でも 6/7ペア通算プラス、ボラフィルタ時は EURUSD/USDCHF/USDCAD で年次中央 PF 1.5–2.0。

## 3. 目標との差(正直な現状)

### 残る壁: 2024年
2024 はボラ急変でなく「通常ボラでの持続的方向(トレンド)」で平均回帰が負ける年。ボラフィルタでは取れず、
これを取れる順張りエッジはメジャー/クロスFXに存在しない(02・03で実証)。
→ **「全年100%プラス・常時PF2.0」は引き続き構造的に到達不能**。ただし 91%/PF1.2 は正直な実運用候補水準。

## 4. 次の探索候補(制約維持のまま)
1. **ボラフィルタの頑健性確認**(vol_pct 0.6–0.8 の高原性 / OOS・ウォークフォワード)。0.70 が偶然でないか。
2. **対象ユニバースの最適な等加重 vs ボラ等価加重**(リスクパリティで年次をさらに平準化)。
3. **2024 の中身を `extract_trades.py` で解剖**し、トレンド検出による「負け回避(撤退)」フィルタを追加(プラス化でなく損失圧縮)。
4. **多時間足合流**(D1 の行き過ぎ + H4 エントリー)で 1対象あたりの PF を底上げ。
