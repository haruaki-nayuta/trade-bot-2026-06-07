# 31. USDJPY 短期(H1)モメンタム = チャンピオンを補う検証済み相補スリーブ(初の成功)

日付: 2026-06-13
対象: `confluence_meanrev_v2`(H4平均回帰)+ 補完候補 = 短期足トレンド
発端(ユーザー): 「検証は長い足中心だった。もっと短い時間足でトレンドを求められないか? 今度はスプレッド負けするか?」

## 結論(先出し)

**通貨内で初めて全ての敵対検証を生き残った相補手法 = USDJPY H1 の ~1日タイムシリーズ・モメンタム。**
reports/30 で「通貨内に底上げ補完なし」と結論したが、それは**長い足(D1中心)の話**だった。**短い足(H1)に降りると別物のエッジが現れる**:

- D1の長期トレンドは GROSS でも負(reports/30)。だが **H1の~1日モメンタムは GROSS Sharpe +0.78**(USDJPY)。
- これは **drift-beta でない真の双方向モメンタム**(ドリフト完全除去後も Sharpe −9%のみ・long/short対称)。
- チャンピオンと **月次相関 −0.10〜−0.15(負)**、**失血窓で稼ぐ**(2022 +1332)。
- 厳密 two-book robust(mm_lab, p95=20%較正)で **robust CAGR +0.9〜+2.3pp(w=0.15〜0.40)・レバ偽装でない**
  (混ぜると経験的DDが縮む=偽装署名の真逆、5シード/ブロック長/IS→OOS 全て純増)。

**= 負相関 × 失血窓で稼ぐ × robust CAGR純増 × レバ偽装なし × 真のエッジ を全て満たす初の候補**。
正直な限界(運用条件): ①単一通貨(USDJPY)集中 ②執行タイミング感応(long-only か ECN 前提)③H1中心。

---

## 1. 発見 — 時間足を下げるとトレンドの符号が変わる

| 検証した足 | ~1日相当lookback の GROSS(コスト0)| 性質 |
|---|---|---|
| D1(reports/30) | 全トレンド族で負(tsmom -0.09 等) | 長期トレンド不在 |
| **H1(本報告)** | **USDJPY +0.78 / GBPJPY +0.28** | 短期モメンタム実在 |
| M5/M15/M30 | グロス正だがコストで全滅 | 回転過多 |

短期足FXは「反転支配」(reports/12)が通説だが、**~1日(H1 lb24)horizonには順張りモメンタムが残る**。
ただし NET はコストに敏感で、M5/M15/M30 はスプレッド負け、**H1 で初めて net が顔を出す**(ユーザーの「スプレッド負けするか」が的中=H1が損益分岐の境界)。

## 2. 広がりと正体 — 「JPY全般」でなく USDJPY中心、だが真のモメンタム

**広がり(`exp_jpy_momentum_robustness`)**: H1 lb24 GROSS Sharpe FULL/IS/OOS:
USDJPY +0.78/+0.34/+1.23(両期✓)、GBPJPY +0.28/+0.25/+0.33(両期✓・弱)、EURJPY −0.03(OOS負)、AUDJPY −0.67。
**「JPY全般のモメンタム」ではない**。EURUSD/GBPUSD は IS のみ正で OOS 負=「EUR/JPY/GBPバスケット」が効いて見えたのは
時間平均のトリック(EUR/GBPがIS・USDJPYがOOSを担当)。**頑健な核は USDJPY**(lookback{12,24,36,48} 全正の高原)。

**drift-beta 監査(`driftbeta_tsmom_usdjpy`)= genuine_momentum**: USD/JPY の +3.87%/年 上昇ドリフトを
**信号・評価の両方から完全除去**しても GROSS Sharpe +0.662→**+0.604(−9%のみ)**。pure drift-beta なら 0 付近に崩壊する。
除去後 long+0.44/short+0.42 と**ほぼ対称・両側個別有意**(t≈1.3-1.4)、純ポジ 53.5%L/46.4%S(ちゃんとドテン)。
年次PnLとbuy&hold方向の符号一致 6/11・corr+0.074≈0(ドリフトベータ署名 不該当)。USD/JPY下落年5回中4回プラス、
最大下落年2020はショート寄与で第2位益。**生の long偏重(+0.96 vs +0.19)はドリフトの見かけで、本質は双方向モメンタム**。

## 3. 補完価値 — 厳密 two-book robust で +0.9〜+2.3pp、レバ偽装でない

`exp60_twobook_robust` + `exp60b_adversarial`(mm_lab, 日次ブロックブートストラップ p95=20% 較正):
champion d1 と USDJPY tsmom を**独立ブック**として各々robust較正→ w で合成→ p95=20% に再レバ。

| w(tsmom配分) | 合成 robCAGR | Δ vs champion(+20.6%) | 経験的maxDD | 最悪年 |
|---|---|---|---|---|
| 0.00 | 20.59% | — | −14.8% | +4.0% |
| 0.15 | 21.51% | **+0.91pp** | | |
| 0.20 | 21.85% | **+1.26pp** | | |
| 0.25 | 22.17% | +1.58pp | | |
| 0.40 | 22.91% | +2.32pp(内点最大) | −12.6% | +6.0% |

**レバ偽装でない決定的証拠**: ①混ぜると経験的maxDDが −14.8%→−12.6% に**縮む**(リスクが増えるどころか減る)。
②same-tail署名テスト(固定レバで混入)は empCAGR↓(+20.6→+12.9%)+p95↓(20→11.9%)=「empCAGR↑+p95悪化」の偽装署名の**真逆**。
CAGR上昇は純粋に「負相関で縮んだリスク予算にレバを盛れる」から。③5シード +2.3〜2.9pp/ブロック長 +2.2〜3.8pp/
IS較正→OOS素検証(+30.5→+33.0%)全て純増。tsmom単独 robCAGR は +1.4% と弱いが、**相関−0.10 のディバーシファイアー**として効く。

## 4. 正直な限界(=運用条件)

1. **単一通貨集中(`exp_plateau_concentration`)**: net 正は実質 USDJPY のみ(GBPJPY弱く追随)。USDJPY を抜いた等加重ブックは
   Sharpe −0.76。n=1 通貨=10年単一標本の集中リスク。構造的根拠(USD/JPY は金利差・JPYファンディング・BoJで最トレンド)は
   あるが要モニタ。GBPJPY 追加は corr+0.24 で DD を緩和するが Sharpe 改善は僅少(GBPJPY も USDJPY建てクロス)。
2. **執行タイミング感応(`exp_tsmom_execution`)**: エッジの約半分がバー内。1バー遅延執行で GROSS Sharpe 0.78→0.45。
   両建て×リテール0.7pip×遅延は NET ≒ 損益分岐。**→ long-only スリーブ(遅延×リテールでも +22.5%)か ECN(0.2pip)前提**。
   損益分岐スプレッド 即時1.30pip。年529取引(日次2件未満)で bot 可。short はコストで死に脚=実質ロング寄り。
3. **H1中心**: M30 は band 必須、H4 は lookback36/48 で崩れる。H1 で運用。

## 5.5. トレンド側の精錬(ユーザー発案: あと+0.5pp 押せるか)— Workflow 6エージェント

w=0.2 を新チャンピオン構成として、モメンタムスリーブ側の改善で増分を測定(全て baseline=USDJPY lb24 w0.2 比、leverage偽装/plateau/OOS/5シード検証つき):

| 改善 | 増分(matched w0.2) | 全監査 | 性質 |
|---|---|---|---|
| **A: lookbackアンサンブル (12,24)** | **+0.57pp**(5シード+0.57〜0.98) | 偽装なし/OOS○ | **両建て・負相関の分散性を保つ**(採用)。ただし高原は(12,24)限定=長LB追加で劣化 |
| C: vol-targetサイジング | +0.50pp(24セル正) | 偽装なし/OOS○ | 分散性保つがwindow感応がU字=限界的 |
| D: long-only化 | +0.73pp | 偽装なし/OOS○ | **性質変化**: USD/JPYと corr+0.73=円安持続前提の方向性ベットに化ける(分散と計上不可) |
| B: GBPJPY 2通貨化 | 棄却 | — | GBPJPYは負け筋(CAGR−0.97%)・USDJPYとcorr+0.26で同一w純減=「悪い分散」 |

**採用 = A(lookbacks=(12,24))**: モメンタムbook単独 robust CAGR +1.43%(lb24)→+2.87%([12,24])、チャンピオン相関−0.10不変=
両建て双方向モメンタムの性質を保ったまま **w=0.2 で +0.57pp 上積み(合計 +1.83pp、合成robCAGR ~22.4%)**。本番 `strategies/usdjpy_momentum_ensemble.py`。
**限界**: 「アンサンブルが一律に効く」ではない((12,24,48)=−0.74pp、(24,48,72)=−0.27pp)=短端2本のconfig特異で高原でない。
効果量+0.5〜0.8ppは robust だが significance の縁。**「lb24→(12,24)の限定差し替え」**として採用、増し盛り(w↑/本数追加)はしない。
D(+0.73pp)は数字最大だが drift-beta化(短脚の負相関=DD縮小効果を捨て、USD/JPYロング・ベットに変質)で**性質を失う**ため非採用
(実弾でshortが死に脚なら long-only は妥当だが、その時は「方向性book・w≤0.2」と明記し分散エッジとして計上しない)。

## 6. 含意

reports/29-30 の「FX分散で底上げ不能」は**長い足・per-instrument/xsec の話**で、**短い足の単一通貨モメンタムという別経路**は閉じていなかった。
これがチャンピオンの天井(robust +18〜20%)を **+1〜2pp** 押し上げる初の正当な相補スリーブ。+10%級ではない(単一通貨ゆえ)が、
**レバでなく真の負相関分散で利益を底上げし、失血窓(2022型USDトレンド)を守る**=ユーザー目標「足を引っ張らず確実に利益を底上げ」を満たす。
次手: ①long-only/ECN前提のデモフォワードで執行劣化を実測 ②GBPJPY 補助ブックの是非 ③単一通貨集中の継続モニタ。
プロトコル追加: **トレンド検証は時間足horizonを掃く**(D1で死んでもH1で生きる)・**短期足エッジは必ずgross/net+ロールオーバー除染+1バー遅延執行の3点で測る**。
