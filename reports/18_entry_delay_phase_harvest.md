# 18. 収益+10%チャレンジ第2ラウンド(金以外) — 採用: エントリー1バー遅延(d1) / 棄却: 位相採集・深い遅延

日付: 2026-06-12 / 対象: `confluence_meanrev_v2` + z-power P=4.0 mp8([[14]][[16]]) / 制約: **金を含む銘柄は使わない**

**問い(ユーザー)**: チャンピオン手法の収益をさらに10%以上改善せよ(金銘柄の取引以外の方法で)。

**結論(先出し)**: 達成。採用した変更は **「エントリーをシグナルバーの次の H4 バー close まで 1 本待つ」(d1) のみ**。
同一テール基準で robust(ブート p95 DD=20%較正, シード0-4平均) **+16.41% → +18.63%(+13.5%相対)**、
empirical 20% 較正でも **+24.64% → +27.50%(+11.6%相対)**、emp 較正時の理論テール p95 は **-27.8% → -27.3% と改善**
= レバ偽装ではない。敵対検証3本(独立再実装/M1粒度DD/機構・近傍)を全通過し、**M1 掛け目込みの実効 CAGR でも
+15.80% → +17.72%(+12.2%相対)** で合格ラインを超えた。本番実装は `strategies/confluence_meanrev_v2_d1.py`。
併せて、有望に見えた**位相オフセット採集の口座レベル崩壊**という重要な負の知見を確定した(§3)。

  全実験がこの値の再現一致を検算してから比較した(exp46/47/51/52/53 全てで一致)。
- 合格ライン: robust 5シード平均 **≥ +18.05%**(+10%相対)+ 6ゲート
  (ペアシード/レバ偽装署名/IS較正→OOS/全年プラス/単年依存禁止/容量健全性)。
- 候補は2系統: ①位相オフセット採集(H4グリッドの恣意性を突く) ②エントリー遅延(reports/15 のコスト監査に
  埋もれていた「1バー遅延で+2.7%」の正式レバー化)。

## 1. 採用 — エントリー1バー遅延 d1(exp47 → 敵対検証 exp51/52/53)

### ルール(本番実装 = `strategies/confluence_meanrev_v2_d1.py`)
v2 がバー t でエントリーシグナル → **t+1 の H4 バー close で建玉**。ただし t+1 時点で z がすでに exit 域へ
戻っていたら(ロング z>-0.5 / ショート z<0.5)**見送り**。exit ルール・サイジング(z_entry=シグナル時点の|z|)は不変。
全て確定バーの値のみ=先読みなし。実装はシグナル変換2行(`le.shift(1) & (z <= -exit_z)` 等)。

### 機構(reports/15 の解剖がそのまま予言していた)
エントリー直後 1〜3 本は平均で逆行する(条件付き収束プレミアムの「第1波」)。1本待つことで:
- (a) 平均的に僅かに有利な価格で建玉(プール段 +2.8%、消滅トレードはわずか7件)
- (b) **逆行第1波が建玉前に起きる=その間は無ポジ**。MtM パスの含み損が浅くなり、p95=20% 較正の k が +7.6% 上がる。
  **利得の主因はこちら**(固定 k 分析 §1-3 で実証)。

### 数値(プロトコルフル)
| 指標 | ベース(d0) | d1 | 差 |
|---|---|---|---|
| robust 5シード平均 | +16.41% | **+18.63%** | +2.22pp(シード別 +1.84〜+2.52、全正) |
| empirical 20% | +24.64%(k=8.27) | **+27.50%**(k=8.90) | +2.86pp |
| emp時 p95 | -27.8% | **-27.3%** | 改善(ブートシード0-9で署名 0/10、p95 は 10/10 で浅い) |
| IS較正→OOS(rob) | +21.04% | **+24.79%** | OOS DD -11.5% vs -10.4%(予算内) |
| 最悪年(emp較正) | +1.8% | **+5.2%** | 全年プラス維持 |
| M1掛け目込み実効CAGR | +15.80%(k×0.965) | **+17.72%(k×0.955)** | +1.92pp(+12.2%相対) |

### 敵対検証(3エージェント並列、全 confirmed)
1. **独立再実装(exp51)**: シグナルレベル実装+vectorbt 約定・コスト経路(exp47 の事後再構成方式とは完全独立)で
   トレード集合 1207/1207 一致(ret 差 maxabs 5.75e-06)。口座値 ±0.14pp 以内で再現。レバレッジ監査:
   emp_k=8.89 で総同時露出 max 14.7x / 1玉 max 3.2x = 国内 25x 規制内。prefix-consistency(先読み)PASS。pytest 26 passed。
2. **M1粒度DD監査(exp52、mp11 を殺したゲート)**: M1 真の谷比は d1 = 1.047〜1.053(d0 1.036)で
   ゲート 1.15 を大差で通過(mp11 は 1.161 で死亡)。深掘りエピソードの顔ぶれ・件数は d0 と同等=d1 固有の
   intrabar テール構造化なし。**掛け目 k×0.955** 適用後も優位の 85〜87% が残存。
   マッチドペア MAE 監査: M1 真値の MAE 改善(+0.035pp)≧ H4 視点の改善(+0.032pp)=
   **「観測を H4 close から外しただけ」仮説を直接棄却**(遅延は逆行第1波を本当に回避している)。
3. **機構・近傍頑健性(exp53)**: パラメータ近傍 OAT 10構成(window/entry_z/exit_z/slow_z/er_max ±)で
   d1−d0 は **10/10 構成で正**(平均+1.64pp、最小+0.20pp)=「d=1 の1点フィット」棄却。
   **固定 k=8.27**(較正上振れの希釈を排除)でも CAGR +0.75pp かつ MtM DD 1.31pp 浅く、p95 も全シードで
   2.1〜2.3pp 浅い=**同一レバでリターン・リスク両面の優位。k 上振れは「稼いだ」もので偽装ではない**。
   MAE 分布は全分位で浅化(p50 -0.369→-0.327%、浅瀬 713→740件)=reports/15 の機構と整合。

### 用量規律: **d≥2 は禁止**(P>4.5 禁止と同型)
- d2: robust +20.8% に見えるが **レバ偽装署名**(emp↑かつ p95 0.92pp 悪化、ブートシード2/3)+
  **プール段利得の88%が2020年集中**(最良年除外で残存12%)→ reject。
- d3: robust +21.3% だが最良年除外で**符号反転**(残存-13%)+プール段 OOS 差分が負+h20 汚染バーに利得の113%集中 → reject。
- **IS-argmax の罠**: IS(<2022)だけ見た研究者は d=3 を選ぶ(IS rob +22.97%)。その d3 はゲートで死ぬ。
  d1 採用は「ゲートを通る最大用量」の線引きであり、口座段の年次残存率(d2 でも73%)は較正 k が全年に乗る
  希釈効果で判別力がない=**単年依存ゲートはプール段で測ること**(プロトコル教訓)。
- 用量反応はハンプ(rob_s0: d0 16.4 → d1 18.2 → d2 20.7 → d3 21.4 → d4 20.1 → d5 16.9 → d6 16.3)=端逃げではない。

### 正直な注記(残課題)
- プール段の価格改善サブチャネル(+2.8%)は**時代依存**: 2016-18 は負(-0.024)、2020-26 は7年連続正。
  保守的に価格チャネルをゼロ評価しても DD 形状チャネル ≈+1.5pp が残る(exp53 時代4分割)。
  運用では**年次の d1−d0 プール差分の符号モニタ**を推奨。
- h20 除染付き d1(遅延先が 20:00 バーならもう1本送る)は rob +18.86% と僅かに上だが、ルール最小主義で不採用(任意装備)。

## 3. 棄却 — 位相オフセット採集(exp46): プール+23%が口座−16〜−28%に反転

**仮説**: H4 グリッド(UTC 0/4/8/12/16/20時)は恣意的な位相であり、サンプリング間隔内に発生して戻る鋭い乖離を
構造的に見逃す。+1h/+2h/+3h オフセットグリッドで同戦略を走らせ、因果 greedy(先にシグナルを出したグリッドが取る、
一玉/銘柄)で統合すれば取り逃しを収穫できるはず。

**プール段は仮説どおり**: ユニークエピソードは実在し高品質(V_all で 646件 +1.9165、平均+29.7bps、
ロールオーバー除染後も残存、全年プラス、ロング/ショート均衡)。統合プールは n=1846 / sum +2.3437(+22.8%)。

**口座段で完全崩壊**: V_all robust 5シード平均 **+11.88%(−27.6%相対)** / V_02 +13.78%(−16.0%相対)。全シードでベース未満。機構は2つ:
1. **置換コスト**: 早い位相が「未成熟な乖離」を浅い段階で先取りし、ベースの良トレード757件(+1.4426)を
   ほぼ損益ゼロの置換コホート743件(−0.0388)で潰す(置換差 **−1.48** がユニーク利得+1.92をほぼ食い潰す)。
   遅く入るほど良い(=d1 が効く)のと同じ機構の裏面: **位相統合は実質「エントリーを早める」変更**だった。
2. **相関同時建玉の増加**(avg_conc 2.99→3.80)で MtM DD あたりのリターンが悪化し、較正 k が 8.27→5.21 に強制低下。
   プール増益≠口座増益(reports/10 の補完エッジ、reports/16 の25銘柄化に続く3例目)。

**副産物の知見**:
- **位相単独監査**: プール段は G1/G2/G3 とも sum +1.73〜1.86 / PF 1.60〜1.73 とベース同格=エッジは位相捏造ではない。
  ただし口座段は rob_s0 +9.0〜10.9% vs ベース+16.4%、emp 時 p95 -35〜-36%=**0h 位相はテールが薄いという
  口座レベルの位相依存が実在**(プール損益の位相頑健性と口座リスクの位相依存は別物)。
- **オフセットグリッド検証には除染必須**: 位相単独 raw→除染(20:00-22:59 の M1 でプライシングされるバーの
  シグナル抑制)で sum −9〜−13%。ロールオーバーBIDアーティファクト([[12]])は H4 でもオフセット位相なら本物の罠。

## 4. プロトコルへの追加(恒久)

1. **単年依存ゲートはプール段で測る**: 口座段(較正後)の年次差分は k 上振れが全年に均等に乗るため希釈されて
   判別力がない(d2 が口座段73%残存に見えた判例)。
2. **「観測粒度の錯覚」検査**: DD 形状改善を主張するレバーは (a)固定 k での DD/p95 改善 (b)M1 真値 MAE の
   マッチドペア比較 の2点で「測り方が変わっただけ」を排除する(exp52/53 で有効性実証)。
3. **執行タイミング系レバーは用量曲線全体+IS-argmax を必須化**: IS は最大用量(d3)を選びがちで、それは
   ゲートで死ぬ変種である(P=4.0/P>4.5 と同型の線引きが3例目)。
